# 🏦 Loan Approval AI System

This is my **first project using Streamlit**, built as part of my journey into Machine Learning and AI development.

The goal of this project is to create a simple web application that predicts loan approval using a trained model, while also exploring how to deploy ML models with an interactive interface.

---

## 🚀 Project Overview

This application allows users to input applicant information and receive a prediction on whether a loan should be approved.

The system uses a **Random Forest model** to estimate the probability of approval.

---

## 📌 Features

* 📥 Input form for applicant data
* 🤖 Machine Learning prediction (Random Forest)
* 📊 Probability-based decision
* 📈 Feature importance visualization
* 🎨 Simple UI built with Streamlit

## ⚙️ How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the app:

```bash
streamlit run app.py
```
